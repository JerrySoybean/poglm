## 1 Installation
From the current directory (which includes this `README.md`), run
```
cd poglm
pip install -e .
```

## 2 Tutorial
`demo.ipynb` is a step-by-step tutorial that learn a POGLM using a specific choice of method combination from a generated synthetic dataset.
